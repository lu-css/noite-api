# noite-api
Noite-API application to hold all diagrams

## Development

> Config .env file before running.

- Install dependencies
    ```sh
    dotnet restore
    ```

- Start mongodb in docker
    ```sh
    docker compose up -d
    ```

- Run Server
    ```sh
    dotnet run
    ```

    or with hot reload
    ```sh
    dotnet watch run
    ```
